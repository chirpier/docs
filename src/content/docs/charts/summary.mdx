# Summary view

A summary view is a textual widget that displays a summary of your stream data.

  style={{
    marginTop: "25px",

  }}
>
  ![Summary View](/summary.png)
</div>

## Additional properties

Here are the additional properties you can configure for your summary view:

- Select which summaries you want to display:
  - Total: The total value of data points for a stream in a time window
  - Count: The number of data points for a stream in a time window
  - Average: The mean value for a stream in a time window
