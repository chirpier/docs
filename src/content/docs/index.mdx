# Overview

[Chirpier](https://www.chirpier.co) is the simplest way to monitor your data in real-time. Use Chirpier to capture data in your application - and visualize it immediately in real-time.

There are only two steps to visualising your data with Chirpier. First,
send your simple Event data packet to our API (or use one of our SDKs).
Second, navigate to your dashboard. Start monitoring any of your data streams using just a few lines of code.

Checkout the [Docs](/docs) pages for more information about integrating with Chirpier.

Checkout the [Use Cases](/use-cases) pages on how to use Chirpier.
